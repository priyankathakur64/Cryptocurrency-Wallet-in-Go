Features:

• Generate private-public key pairs using the ECDSA algorithm.
• Create transactions with sender, receiver, and amount details.
• Sign transactions using the sender's private key.
• Verify transaction authenticity using digital signatures.
• Modular structure for extensibility and scalability.

Tech Stack:

• Language: Go (Golang)

Core Libraries:

• crypto/ecdsa: For elliptic curve cryptography.
• crypto/elliptic: For the P-256 curve.
• crypto/rand: For secure random number generation.
• log: For error handling and logging.
